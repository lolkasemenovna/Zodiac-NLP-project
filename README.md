# Zodiac-NLP-project (NLP-анализ характеристик знаков зодиака на женском форуме)
Учебный проект по программе ДПО НИУ ВШЭ "Компьютерная лингвистика"

Подготовила: Дмитриева Анастасия Павловна

## Описание проекта
Нашей целью было изучить характеристики мужчин по знакам зодиака на основе комменатриев с женского форума, а затем сопоставить полученные характеристики с эталонным текстом (стереотипное описание знаков в гороскопах). 
Для этого нам понадобились следующие инструменты и библиотеки. 
## Инструменты и библиотеки

### Сбор данных
- **requests** — отправка HTTP-запросов к сайту woman.ru
- **BeautifulSoup** — парсинг HTML-страниц и извлечение комментариев

### Обработка данных
- **pandas** — хранение и обработка данных в табличном формате
- **re** — очистка текста с помощью регулярных выражений

### Обработка текста (NLP)
- **nltk** — токенизация текста, стоп-слова
- **pymorphy3** — морфологический анализ: лемматизация, определение части речи, рода, падежа и числа

### Статистический анализ
- **collections.Counter** — подсчёт частот слов
- **sklearn (TfidfVectorizer)** — вычисление TF-IDF для выявления ключевых слов каждого знака
- **nltk.collocations** — поиск биграмм и триграмм, подсчёт PMI

### Визуализация
- **WordCloud** — облака слов для каждого знака зодиака
- **matplotlib** — построение графиков
- **seaborn** — тепловые карты (heatmap)

## Как работать с файлами: 
1. Запустите в редакторе кода json-файл "Анализ характеристик знаков зодиака"
2.  Создайте папку `Zodiac` в той же директории, куда установили этот ноутбук
3. Скачайте csv-данные по ссылке: https://drive.google.com/drive/folders/1RwZZL32A5W-TZ4M0UZ5XhV3OiWRv5zKS?usp=sharing
4. Положите все CSV файлы из архива в папку `Zodiac/`
5. Скачайте файл stereotypes.txt из данного репозитория, он должен лежать в той же директории, где установлен этот ноутбук. 
6. Запускайте ячейки по порядку
